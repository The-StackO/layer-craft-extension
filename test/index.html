📋 使用说明

  1. 启动 Layer Craft 扩展
  2. 点击"选择元素"按钮
  3. 将鼠标悬停在任何图片元素上
  4. 点击选择图片元素
  5. 在弹出的面板中输入新的图片URL
  6. 点击确认按钮完成替换

支持的图片类型：

  • ✅ IMG 标签图片
  • ✅ PICTURE 标签图片
  • ✅ CSS 背景图片
  • ✅ CSS Content 属性图片
